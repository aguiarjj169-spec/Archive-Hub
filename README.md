# 🚀 Archive Hub

O **Archive Hub** é um script de Roblox focado em estabilidade, performance e ferramentas essenciais para o seu jogo. Sem firulas, apenas o que funciona.

> **⚠️ NOTA IMPORTANTE:** Este script foi desenvolvido **exclusivamente para PC**. Algumas funcionalidades (como Autoclicker e comandos por teclado) não são otimizadas ou não funcionam em dispositivos mobile/celular.

## 📋 Funcionalidades
* 🎯 **Aimbot:** Sistema de mira relativa (mousemoverel) para maior precisão.
* 👁️ **ESP:** Identificação visual de jogadores.
* 📦 **Hitbox:** Ajuste de tamanho para facilitar o combate.
* ✈️ **Fly:** Sistema de voo otimizado.
* 🛡️ **God Mode:** Proteção básica ativada.
* 🖱️ **Autoclicker:** Ferramenta de clique automático (exclusivo para PC).
* 📍 **Click Teleport:** Teleporte simples ao clicar com o controle pressionado.
